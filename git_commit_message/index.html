2 Types list

  • ✨ feat: add new feature
  • 🐛 fix: fix bug
  • 🚧 work-in-progress:
  • 🚑️ hotfix:
  • ✏️ fix-typos:
  • ♻️ refactor:
  • 📝 docs: documents
  • chore: small bugs not-relevant to code
  • 🎨 style
  • ⚡️ perf: performance
  • vendor: Update dependencies’, packages’ … version …

Gitemoji link: https://gitmoji.dev/

3 Notes:

  • BREAKING CHANGE in body.
