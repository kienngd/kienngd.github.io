PHP “pass by value” VS “pass by reference”

     Kien Nguyen -    2021-16-04
    notes
    PHP    

Hello, it is me again. Today, we will talk about “pass by value” or “pass by reference”. I use PHP 7.4 and Ubuntu 20.04 to test. Happy coding!

We have an example class:

class MyClass {
   public $x;
   public $y;
 
   public function __construct($x, $y) {
       $this->x = $x;
       $this->y = $y;
   }
 
   public function setX($x)
   {
       $this->x = $x;
   }
 
   public function setY($y)
   {
       $this->y = $y;
   }
}

Get token
curl -s -i -H "Content-Type: application/json"   -d '
{
    "auth": {
        "identity": {
            "methods": [
                "password"
            ],
            "password": {
                "user": {
                    "name": "{USERNAME}",
                    "domain": { "id": "{DOMAIN_ID}" },
                    "password": "{PASSWORD}"
                }
            }
        },
        "scope": {
            "project": {
                "id": "{PROJECT_ID}"
            }
        }
    }
}'   "https://{BASE URL}:5000/v3/auth/tokens" | grep "X-Subject-Token:"

Default keystone port is 5000

By default, this token will be expired after 6 hours.
